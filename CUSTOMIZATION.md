# 🔧 Customizing CasinoEconomyBot

This bot can be easily customized if you want to use your own bot name, branding, or creator information. You are free to modify all text, embeds, and labels to match your project.

## 📝 Changing the Bot Name
Some commands contain the name **CasinoEconomyBot** inside:
- embed titles
- descriptions
- footers
- system messages

To use your own bot name, simply replace all occurrences of:
```
CasinoEconomyBot
```
with your preferred name, for example:
```
MyDiscordBot
```

## 👤 Changing the Creator Name
Some files may include text such as:
```
Created by: Sk8rfu
```
or
```
Developer: Sk8rfu
```

You can freely change this to your own name or remove it entirely.

## 📁 Where to Find These Names
You will usually find these references in:
- `/commands/` (inside embeds)
- `about.js`
- `help.js` (if included)
- footers in various commands

## 🔍 How to Find and Replace Quickly
If you use VS Code:
1. Press **Ctrl + Shift + F**
2. Search for:
```
CasinoEconomyBot
```
3. Edit all results as needed.

## ✔ Free to Modify
You are allowed to change:
- the bot name
- the creator name
- embed messages
- command responses
- visual style
- branding

This project is fully customizable.
